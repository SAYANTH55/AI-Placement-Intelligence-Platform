# AI Driven Placement Intelligence Platform - Team Roles

## Member 1: Frontend Developer (UI & UX)
- **Responsibility**: Designing and implementing the user interface for both students and teachers.
- **Key Tasks**:
    - Build Student Dashboard & Portal.
    - Build Teacher/Admin Dashboard & Analytics.
    - Implement Resume Upload & Result Visualization.
- **Tech Stack**: React.js, Tailwind CSS.

## Member 2: Backend Developer (API & Database)
- **Responsibility**: Building the robust server-side infrastructure and managing data.
- **Key Tasks**:
    - Develop FastAPI/Flask RESTful APIs.
    - Implement Database Schema (MySQL/MongoDB).
    - Integrate Frontend with AI Engine.
- **Tech Stack**: Python, FastAPI, MySQL/MongoDB.

## Member 3: AI/ML Engineer (Intelligence Layer)
- **Responsibility**: Developing the core AI models and NLP processing units.
- **Key Tasks**:
    - Resume Parsing & NLP (spaCy/NLTK).
    - Skill Gap Analysis & Job Matching.
    - Placement Prediction Modeling (scikit-learn).
- **Tech Stack**: Python, scikit-learn, pandas, spaCy/NLTK.
