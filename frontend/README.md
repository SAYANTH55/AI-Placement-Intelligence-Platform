# Frontend Module (UI & UX)

Designed for two main portals:
1. **Student Portal**: Dashboard and resume upload.
2. **Teacher Portal**: Admin dashboard and analytics.

## Structure
- `student-portal/`: Specialized components for student view.
- `teacher-portal/`: Specialized components for teacher/admin view.
- `components/`: Shared UI components (Buttons, Inputs, Modals).
- `pages/`: Parent page structures.

## Getting Started
- Built with React.
- To start development: `npm install` (after initializing with `create-react-app` or `vite`).
